PyPong  
======  

Pong in Python  

Mac OS X Install using Homebrew:  
  Tested on Snow Leopard.  
  
  1. Homebrew  
    Homepage: http://mxcl.github.com/homebrew  
    Install command:  
      ruby <(curl -fsSkL raw.github.com/mxcl/homebrew/go)  
  
  2. Python 2.7.3  
    Homepage: http://www.python.org  
      Homebrew and Python Info: https://github.com/mxcl/homebrew/wiki/Homebrew-and-Python  
    Install command:  
      brew install python  
    
  3. PyGame  
    Homepage: http://www.pygame.org  
    Install command for required libraries:  
      brew install sdl sdl_image sdl_mixer sdl_ttf smpeg portmidi  
    Install command for PyGame:  
      /usr/local/share/python/pip install hg+http://bitbucket.org/pygame/pygame  
      
  4. Play:  
    In Terminal, change to directory with PyPong source and sound files. Then enter:  
      python PyPong.py  

Windows Install:  
  Tested on XP  
  
  1. Python 2.7.3  
    Homepage: http://www.python.org  
    Install:  
      http://python.org/ftp/python/2.7.3/python-2.7.3.msi  
    
  2. PyGame  
    Homepage: http://www.pygame.org  
    Install:  
      http://pygame.org/ftp/pygame-1.9.1.win32-py2.7.msi  
    
  3. Play  
    Open command prompt.  
    Change to directory with PyPong source and sound files.  Then enter:  
      c:\Python\Python.exe PyPong.py
